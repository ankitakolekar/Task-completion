Day 1 – Building My First Web Page

On Day 1, I learned how to build my very first web page using HTML for structure and CSS for styling.

What I Learned

  • Basic HTML structure – <html>, <head>, <body>
  • Headings, paragraphs, and links
  • Linking CSS to HTML
  • Styling text and background using CSS

Example Code

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage using HTML and CSS.</p>
  </body>
</html>

“It was my first step into web design. I learned how HTML gives structure and CSS makes it beautiful.”

Day 2 – Getting to Know HTML

Today I explored HTML semantics, block vs inline elements, and how to use tags correctly to give meaning to web content.

Key HTML Concepts

  • Semantic tags like <header>, <nav>, <section>, <footer>
  • Using <div> and <span> for structure and styling
  • Creating internal and external links with <a>
  • Encoding special characters like © and é

Example Website Structure

<header>
  <h1>Styles Conference</h1>
  <nav>
    <a href="#">Home</a>
    <a href="#">Speakers</a>
    <a href="#">Schedule</a>
  </nav>
</header>
<footer>
  <small>© Styles Conference</small>
</footer>

“I now understand how each HTML tag gives meaning to content and helps in SEO and accessibility.”

Day 3 – Getting to Know CSS

On Day 3, I learned how CSS works through the cascade, specificity, and selectors. I also explored color values and length units.

Core CSS Topics

  • The Cascade – last rule wins
  • Specificity – ID > Class > Type
  • Combining selectors (e.g., .box p)
  • Multiple classes for modular design
  • Color formats: HEX, RGB, RGBA, HSL
  • Units: px, %, em for responsive design

Example CSS Code

p {
  color: blue;
}
p {
  color: red; /* Last rule wins */
}

#main {
  background: #f4f4f4;
}
.container p.note {
  color: rgba(255, 0, 0, 0.8);
}

“CSS makes web pages come alive! I learned how specificity and selectors decide which style wins.”
